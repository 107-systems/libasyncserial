:floppy_disk: `libasyncserial`
==============================
A C++ serial interface library based on `boost::asio`.

### How-to-build
```bash
git clone https://github.com/107-systems/libasyncserial && cd libasyncserial
mkdir build && cd build
cmake .. && make
```
or
```bash
cmake -DBUILD_EXAMPLES=ON .. && make
```
